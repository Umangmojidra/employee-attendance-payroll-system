# Employee Attendance & Payroll Management System 🧾

A console-based Python application for managing employee attendance, payroll, and department-wise salary analytics using **MySQL**.

## 🔧 Technologies Used
- Python
- MySQL
- mysql-connector-python
- Tabulate
- CSV

## 📌 Features
- Employee CRUD operations
- Daily attendance tracking
- Late login penalty handling
- Payroll generation with tax, PF, bonus & overtime
- Salary slip generation
- Attendance reminders
- Department-wise payroll summary export (CSV)
- Manager authentication system

## 🗄 Database
The system uses a MySQL database with tables such as:
- employees
- attendance
- payroll
- departments
- manager

## ▶️ How to Run
1. Clone the repository
2. Install dependencies  

pip install -r requirements.txt

3. Configure MySQL credentials in `main.py`
4. Run the program  

python main.py
